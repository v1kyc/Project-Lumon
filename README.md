# Lumon

A fast, lightweight TUI framework for Java 21.

> ⚠️ Work in progress — not ready for production use.

## Vision

Lumon is being built around a simple blocks-and-buttons component model,
with a focus on minimal memory footprint and zero external dependencies.

## Status

This project is in early design/architecture phase. No stable API yet.
